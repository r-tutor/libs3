Olkin95 {meta}R Documentation

Thrombolytic Therapy after Acute Myocardial Infarction

Description

Meta-analysis on Thrombolytic Therapy after Acute Myocardial Infarction

Format

A data frame with the following columns:

author first author
year year of publication
event.e number of events in experimental group
n.e number of observations in experimental group
event.c number of events in control group
n.c number of observations in control group

Source

Olkin I (1995): Statistical and theoretical considerations in meta-analysis. Journal of Clinical Epidemiology, 48, 133–46

Examples

data(Olkin95)
summary(metabin(event.e, n.e, event.c, n.c, data = Olkin95))
