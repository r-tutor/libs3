Fleiss93 {meta}R Documentation

Aspirin after Myocardial Infarction

Description

Meta-analysis on aspirin in preventing death after myocardial infarction.

Data example in Fleiss (1993) for meta-analysis with binary outcomes.

Usage

data(Fleiss93)

Format

A data frame with the following columns:

study

Study label

year

Year of publication

event.e

Number of deaths in aspirin group

n.e

Number of observations in aspirin group

event.c

Number of deaths in placebo group

n.c

Number of observations in placebo group

Source

Fleiss JL (1993), The statistical basis of meta-analysis. Statistical Methods in Medical Research, 2, 121–145.

Examples

data(Fleiss93)
metabin(event.e, n.e, event.c, n.c,
        data=Fleiss93,
        studlab=paste(study, year),
        sm="OR", comb.random=FALSE)
