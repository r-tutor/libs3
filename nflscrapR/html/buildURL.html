buildURL {nflscrapR}R Documentation

Building URL to scrape player season stat pages

Description

This is a sub-function for the season_rosters function.

Usage

buildURL(position, season = 2016, page = 1, type = c("REG", "POST",
  "PRE"))

Arguments

position:

(character string) Specifies a player position page for the URL

season:

4-digit year associated with a given NFL season

page:

1-digit page number to look into

type:

A three character string specifying the season type
