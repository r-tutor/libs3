wgts {MAc}R Documentation

Weights added to Meta Data

Description

Adds weights to the meta-analysis data set.

Usage

wgts(es, var.es, data)

Arguments

es

r or z' effect size.

var.es

Variance of es

data

data.frame with values above.

Value

Adds fixed and random-effects weights and confidence intervals to meta data set.
