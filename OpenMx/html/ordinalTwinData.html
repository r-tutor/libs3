ordinalTwinData {OpenMx}R Documentation

Data for ordinal twin model

Description

Example data for ordinal twin-data modelling. Three variables measured in each twin.

Usage

data("ordinalTwinData")

Format

A data frame with 139 observations on the following 7 variables.

zyg

a numeric vector

var1_twin1

a numeric vector

var2_twin1

a numeric vector

var3_twin1

a numeric vector

var1_twin2

a numeric vector

var2_twin2

a numeric vector

var3_twin2

a numeric vector

Examples

data(ordinalTwinData)
str(ordinalTwinData)
