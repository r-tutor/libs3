myRegData {OpenMx}R Documentation

Example regression data with correlated predictors

Description

Data set used in some of OpenMx's examples.

Usage

data("myRegData")

Format

A data frame with 100 observations on the following variables.

w

Predictor variable

x

Predictor variable

y

Predictor variable

z

Outcome varialbe

Details

w, x, and y are predictors of z. x and y are correlated.

Source

Simulated.

References

The OpenMx User's guide can be found at http://openmx.ssri.psu.edu/documentation.

Examples

data(myRegData)
summary(lm(z ~ ., data=myRegData))
