mxComputeDefault {OpenMx}R Documentation

Default compute plan

Description

The default compute plan is approximately as follows: mxComputeSequence(list(mxComputeGradientDescent(), mxComputeConfidenceInterval(), mxComputeNumericDeriv(), mxComputeStandardError(), mxComputeReportDeriv()))

Usage

mxComputeDefault(freeSet = NA_character_)

Arguments

freeSet

names of matrices containing free variables
