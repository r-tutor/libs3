val_labels_to_na {labelled}R Documentation

Recode value labels to NA

Description

For labelled variables, values with a label will be recoded to NA.

Usage

val_labels_to_na(x)

Arguments

x

Object to recode.

...

Other arguments passed down to method.

See Also

zap_labels

Examples

v <- labelled(c(1, 2, 9, 1, 9), c(dk = 9))
val_labels_to_na(v)
